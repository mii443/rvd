# RISC-V Debugger
[rsrv](https://github.com/morioka22/rsrv) based RISC-V debugging tool.

# Installation
```sh
cargo install --git https://github.com/morioka22/rvd
```

# Usage
```
Usage: rvd [OPTIONS] <FILE>

Arguments:
  <FILE>

Options:
  -m, --mem <MEM>  [default: 65536]
  -h, --help       Print help
```
